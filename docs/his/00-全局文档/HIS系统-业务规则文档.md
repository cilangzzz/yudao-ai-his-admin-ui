## 3. 规则详细定义

### 3.1 M09 系统管理规则

#### BR-SYS-001: 用户名校验

| 属性 | 内容 |
|------|------|
| 规则编号 | BR-SYS-001 |
| 规则名称 | 用户名格式校验 |
| 规则类型 | 校验规则 |
| 适用对象 | 用户名(username) |
| 规则描述 | 用户名必须唯一，3-20位，仅允许字母、数字、下划线 |
| 触发时机 | 创建/修改用户时 |
| 校验逻辑 | 正则：`^[a-zA-Z][a-zA-Z0-9_]{2,19}$`，且全局唯一 |
| 失败处理 | 返回错误"用户名已存在"或"用户名格式不正确" |
| 优先级 | P0 |

#### BR-SYS-002: 密码复杂度校验

| 属性 | 内容 |
|------|------|
| 规则编号 | BR-SYS-002 |
| 规则名称 | 密码复杂度校验 |
| 规则类型 | 校验规则 |
| 适用对象 | 密码(password) |
| 规则描述 | 密码必须≥8位，含大写字母、小写字母、数字、特殊字符至少三类 |
| 触发时机 | 创建用户/修改密码时 |
| 校验逻辑 | `length >= 8 AND (upper + lower + digit + special >= 3)` |
| 失败处理 | 返回错误"密码不满足复杂度要求" |
| 优先级 | P0 |

#### BR-SYS-003: 登录失败锁定

| 属性 | 内容 |
|------|------|
| 规则编号 | BR-SYS-003 |
| 规则名称 | 连续登录失败锁定 |
| 规则类型 | 时间规则 |
| 适用对象 | 用户账号 |
| 规则描述 | 连续登录失败5次，锁定账户30分钟 |
| 触发时机 | 用户登录时 |
| 校验逻辑 | `IF fail_count >= 5 AND last_fail_time + 30min > now THEN lock` |
| 失败处理 | 返回"账户已锁定，请30分钟后重试" |
| 优先级 | P0 |

#### BR-SYS-004: 角色权限三级控制

| 属性 | 内容 |
|------|------|
| 规则编号 | BR-SYS-004 |
| 规则名称 | RBAC三级权限控制 |
| 规则类型 | 权限规则 |
| 适用对象 | 角色权限 |
| 规则描述 | 权限分为菜单权限、按钮权限、数据权限三级 |
| 触发时机 | 用户访问功能时 |
| 校验逻辑 | `菜单权限→页面可见性, 按钮权限→操作可用性, 数据权限→数据范围` |
| 优先级 | P0 |

#### BR-SYS-005: 审计日志不可篡改

| 属性 | 内容 |
|------|------|
| 规则编号 | BR-SYS-005 |
| 规则名称 | 审计日志只增不改 |
