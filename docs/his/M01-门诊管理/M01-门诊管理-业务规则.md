#### BR-OP-005: 处方CDS四维校验

| 属性 | 内容 |
|------|------|
| 规则编号 | BR-OP-005 |
| 规则名称 | 处方开立CDS自动校验 |
| 规则类型 | 校验规则 |
| 适用对象 | 处方(prescription)、医嘱 |
| 规则描述 | 处方开立时必须进行四维CDS校验：药物相互作用、过敏检查、剂量合理性、配伍禁忌。高风险警告不可忽略，必须修改处方 |
| 触发时机 | 处方/医嘱提交时 |
| 校验逻辑 | ```
// 伪代码
warnings = []

// 1. 药物相互作用检查
FOR each drug1, drug2 IN prescription.drugs COMBINATION(2) DO
    interaction = CHECK_drug_interaction(drug1, drug2)
    IF interaction EXISTS THEN
        warnings.add({
            'type': 'INTERACTION',
            'level': interaction.risk_level,  // HIGH/MEDIUM/LOW
            'message': interaction.description,
            'mechanism': interaction.mechanism,
            'suggestion': interaction.suggestion
        })
    END IF
END FOR

// 2. 过敏史检查
FOR each drug IN prescription.drugs DO
    FOR each allergy IN patient.allergies DO
        IF drug.drug_class CONTAINS allergy.allergen OR drug.ingredients CONTAINS allergy.allergen THEN
            warnings.add({
                'type': 'ALLERGY',
                'level': 'HIGH',
                'message': '患者对' + allergy.allergen + '过敏，' + drug.drug_name + '含有该成分',
                'suggestion': '建议更换为其他类别药物'
            })
        END IF
    END FOR
END FOR

// 3. 剂量合理性检查
FOR each drug IN prescription.drugs DO
    dose_check = CHECK_dose_rationality(drug, patient.age, patient.weight, patient.renal_function)
    IF dose_check.is_abnormal THEN
        warnings.add({
            'type': 'DOSE',
            'level': 'MEDIUM',
            'message': dose_check.message,
            'suggestion': '推荐剂量范围：' + dose_check.recommended_range
        })
    END IF
END FOR

// 4. 配伍禁忌检查（输液处方）
IF prescription.type = 'INFUSION' THEN
    FOR each drug1, drug2 IN prescription.drugs COMBINATION(2) DO
        incompatibility = CHECK_incompatibility(drug1, drug2)
        IF incompatibility EXISTS THEN
            warnings.add({
                'type': 'INCOMPATIBILITY',
                'level': 'HIGH',
                'message': incompatibility.description,
                'suggestion': '不可在同一输液袋中混合，建议分开输注'
            })
        END IF
    END FOR
END IF

// 处理高风险警告
high_risk_warnings = FILTER warnings WHERE level = 'HIGH'
IF COUNT(high_risk_warnings) > 0 THEN
    RETURN block_with_warnings(high_risk_warnings, "高风险警告不可忽略，请修改处方")
END IF

RETURN success_with_warnings(warnings)
``` |
| 失败处理 | 高风险警告：阻止处方保存，强制要求修改；中/低风险警告：显示警告，医生确认原因后可继续 |
| 优先级 | P0 |

#### BR-OP-006: 处方格式校验

| 属性 | 内容 |
|------|------|
| 规则编号 | BR-OP-006 |
| 规则名称 | 处方格式校验 |
| 规则类型 | 校验规则 |
| 适用对象 | 处方(prescription) |
| 规则描述 | 处方必须关联诊断，药品信息完整（药品名称、规格、用法、用量、频次、天数），且符合处方书写规范 |
| 触发时机 | 处方提交时 |
| 校验逻辑 | ```
// 伪代码
// 诊断校验
IF prescription.diagnose_code IS NULL OR prescription.diagnose_code = '' THEN
    RETURN error("处方必须关联诊断，请先开立诊断")
END IF

// 药品信息校验
FOR each item IN prescription.items DO
    IF item.drug_id IS NULL THEN
        RETURN error("药品信息不完整")
    END IF
    IF item.dosage IS NULL OR item.dosage <= 0 THEN
        RETURN error("药品[" + item.drug_name + "]用量不能为空或小于等于0")
    END IF
    IF item.frequency_code IS NULL THEN
        RETURN error("药品[" + item.drug_name + "]用药频次不能为空")
    END IF
    IF item.route_code IS NULL THEN
        RETURN error("药品[" + item.drug_name + "]给药途径不能为空")
    END IF
    IF item.days IS NULL OR item.days <= 0 THEN
        RETURN error("药品[" + item.drug_name + "]用药天数不能为空或小于等于0")
    END IF
END FOR

// 处方类型校验
IF prescription.prescription_type = 'ANESTHETIC' THEN  // 麻醉药品处方
    IF prescription.items[0].quantity > get_anesthetic_limit(prescription.diagnose_code) THEN
        RETURN error("麻醉药品超过限量标准")
    END IF
END IF

RETURN success()
``` |
| 失败处理 | 返回具体校验失败原因，阻止处方保存 |
| 优先级 | P0 |

#### BR-OP-009: 药品库存校验

| 属性 | 内容 |
|------|------|
| 规则编号 | BR-OP-009 |
| 规则名称 | 药品库存校验 |
| 规则类型 | 校验规则 |
| 适用对象 | 处方(prescription)、发药(dispensing) |
| 规则描述 | 处方开立和发药时必须校验药品库存，库存不足时提示并提供替代药品建议 |

| 校验逻辑 | ```
// 伪代码
FOR each item IN prescription.items DO
    stock = SELECT current_stock, safety_stock 
            FROM pharmacy_stock 
            WHERE drug_id = item.drug_id AND pharmacy_id = :pharmacy_id
    
    IF stock IS NULL THEN
        RETURN error("药品[" + item.drug_name + "]不存在或未配置库存")
    END IF
    
    IF stock.current_stock < item.quantity THEN
        // 查找替代药品
        alternatives = SELECT * FROM drug_alternative 
                       WHERE original_drug_id = item.drug_id
        RETURN error("药品[" + item.drug_name + "]库存不足，当前库存：" + stock.current_stock, alternatives)
    END IF
END FOR

RETURN success()
``` |
| 失败处理 | 返回错误"库存不足"，显示当前库存数量，提供替代药品建议 |

#### BR-OP-011: 麻精药品专项校验

| 属性 | 内容 |
|------|------|
| 规则编号 | BR-OP-011 |
| 规则名称 | 麻醉精神药品专项校验 |
| 规则类型 | 校验规则 |

| 触发时机 | 麻精药品处方开立时、发药时 |
| 校验逻辑 | ```
// 伪代码
IF prescription.prescription_type IN ('ANESTHETIC', 'PSYCHOTROPIC') THEN
    // 1. 医生处方权校验
    doctor_permission = SELECT * FROM doctor_permission 
                        WHERE doctor_id = :doctor_id 
                          AND permission_type = 'NARCOTIC_PRESCRIPTION'
    IF doctor_permission IS NULL OR doctor_permission.status != 'VALID' THEN
        RETURN error("该医生无麻精药品处方权")
    END IF
    
    // 2. 限量校验
    limit = get_narcotic_limit(prescription.diagnose_code, prescription.prescription_type)
    FOR each item IN prescription.items DO
        IF item.quantity > limit THEN
            RETURN error("麻精药品超过规定限量标准")
        END IF
    END FOR
    
    // 3. 患者身份证校验
    IF patient.id_card_no IS NULL THEN
        RETURN error("麻精药品处方必须登记患者身份证号")
    END IF
END IF

RETURN success()
``` |
| 失败处理 | 返回具体校验失败原因，阻止处方保存或发药 |

| 适用对象 | 退费记录(refund) |
| 规则描述 | 已执行的项目不可退费，退费项目必须是已缴费且未执行状态 |
| 触发时机 | 退费申请时 |
| 校验逻辑 | ```
// 伪代码
FOR each item IN refund.items DO
    payment_item = SELECT * FROM payment_item WHERE item_id = item.item_id
    
    // 已缴费校验
    IF payment_item.payment_status != 'PAID' THEN
        RETURN error("项目[" + item.item_name + "]未缴费，不可退费")
    END IF
    
    // 执行状态校验
    IF payment_item.execute_status = 'EXECUTED' THEN
        RETURN error("项目[" + item.item_name + "]已执行，不可退费")
    END IF
    
    // 发药状态校验（药品）
    IF item.item_type = 'DRUG' THEN
        prescription = SELECT * FROM prescription WHERE prescription_id = item.prescription_id
        IF prescription.prescription_status = 'DISPENSED' THEN
            RETURN error("药品已发放，请先办理退药")
        END IF
    END IF
END FOR

RETURN success()
``` |
| 失败处理 | 返回错误"项目已执行，不可退费"或"药品已发放，请先办理退药"，阻止退费操作 |

#### BR-OP-013: 退药条件校验

#### BR-OP-052: 预约超时处理

| 属性 | 内容 |
|------|------|
| 规则编号 | BR-OP-052 |
| 规则名称 | 预约超时未签到处理 |
| 规则类型 | 时间规则 |
| 适用对象 | 预约记录(appointment) |
| 规则描述 | 预约时间过后30分钟未签到，预约自动过期，号源释放 |
| 触发时机 | 定时检查 |
| 校验逻辑 | ```
// 伪代码
// 定时任务检查
FOR each appointment IN SELECT * FROM op_appointment 
                         WHERE status = 'APPOINTED' 
                           AND appointment_date = CURRENT_DATE DO
    
    // 获取预约时间段结束时间
    time_slot_end = appointment.appointment_date + appointment.time_slot_end
    minutes_elapsed = MINUTE(NOW() - time_slot_end)
    
    // 超过30分钟未签到
    IF minutes_elapsed > 30 THEN
        // 标记为已过期
        appointment.status = 'EXPIRED'
        appointment.expire_time = NOW()
        
        // 释放号源
        UPDATE op_schedule 
        SET remaining_count = remaining_count + 1 
        WHERE schedule_id = appointment.schedule_id
        
        // 发送过期通知
        SEND_notification({
            'patient_id': appointment.patient_id,
            'type': 'APPOINTMENT_EXPIRED',
            'message': '您的预约已过期，号源已释放，如需就诊请重新预约'
        })
    END IF
END FOR
``` |
| 失败处理 | 系统自动标记过期，释放号源，发送通知 |
| 优先级 | P0 |
