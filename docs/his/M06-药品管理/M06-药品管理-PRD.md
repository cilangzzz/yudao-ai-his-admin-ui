## 1. 子系统概述

### 1.1 子系统定位

药品管理子系统是YUDAO-AI-HIS的核心支撑模块，覆盖药库管理、采购管理、库存管理、处方审核、合理用药、特殊药品管理等功能。系统实现药品全流程追溯，支持效期管理和批号追踪，提供处方合理性自动审核（CDS四维校验），实现麻醉药品"五专"管理。

### 1.2 业务目标

| 目标类型 | 目标描述 | 衡量指标 |
|----------|----------|----------|
| 安全目标 | 实现处方合理性自动审核 | CDS审核覆盖率100% |
| 质量目标 | 实现药品效期和批号追溯 | 近效期预警率100% |
| 合规目标 | 麻精药品五专管理 | 合规率100% |
| 效率目标 | 药品库存准确管理 | 库存准确率≥99% |

### 2.2 M06-03 处方审核与合理用药

#### 2.2.1 CDS四维校验

系统实现处方合理性自动审核的四维校验：

**维度1：药物相互作用检查**

检查药物-药物之间的相互作用，包括：
- 药效学相互作用（协同、拮抗）
- 药动学相互作用（吸收、分布、代谢、排泄）
- 不良反应叠加

风险等级：
- 高风险：禁止配伍，必须有替代方案
- 中风险：需要密切监测，有替代方案建议
- 低风险：可以配伍，但需注意监测

**维度2：过敏检查**

基于患者过敏史进行药物过敏风险检查：
- 检查药品是否属于患者过敏药物类别
- 检查药品成分是否含过敏物质
- 交叉过敏预警

**维度3：剂量合理性检查**

基于患者特征进行剂量合理性检查：
- 年龄：老年人、儿童剂量调整建议
- 体重：体重相关剂量计算
- 肾功能：肾功能不全剂量调整
- 肝功能：肝功能不全剂量调整

**维度4：配伍禁忌检查**

静脉用药配伍禁忌检查：
- 物理性配伍禁忌（沉淀、变色）
- 化学性配伍禁忌（分解、失效）
- 理性配伍禁忌

#### 2.2.2 CDS校验流程

```
处方提交 → CDS自动校验
    │
    ├── 维度1: 药物相互作用 ──→ 有 ──→ 显示警告（风险等级+机制+建议）
    │
    ├── 维度2: 过敏检查 ──→ 有过敏 ──→ 显示警告（过敏原+风险+建议）
    │
    ├── 维度3: 剂量合理性 ──→ 异常 ──→ 显示提示（推荐范围+计算依据）
    │
    └── 维度4: 配伍禁忌 ──→ 有 ──→ 显示警告（禁忌类型+原因）
    │
    ↓
校验结果汇总
    │
    ├── 全部通过 ──→ 审核通过 ──→ 进入调配流程
    │
    ├── 有警告 ──→ 显示警告详情 ──→ 药师确认处理
    │               │
    │               ├── 确认继续 ──→ 审核通过
    │               └── 退回修改 ──→ 通知医生
    │
    └── 有高风险 ──→ 强制退回 ──→ 必须修改处方
```

### 2.3 M06-04 特殊药品管理

#### 2.3.1 麻醉药品"五专"管理

| 专人 | 专人负责麻醉药品管理，责任明确 |
|------|------|
| 专柜 | 专用柜存放，与其他药品分开 |
| 专锁 | 专锁保管，双人双锁管理 |
| 专账 | 专用账册登记，逐日消耗记录 |
| 专方 | 专用处方开具，处方编号管理 |

#### 2.3.2 麻醉药品流程

```
麻醉药品处方开具
    │
    ├── 专用处方格式
    ├── 处方编号管理
    └── 医生资质审核
    │
    ↓
处方审核（药师）
    │
    ├── 处方格式审核
    ├── 用药合理性审核
    └── 医生资质确认
    │
    ↓
药品调配（双人操作）
    │
    ├── 开专柜（双人）
    ├── 取药品
    ├── 核对处方
    └── 调配药品
    │
    ↓
发药确认
    │
    ├── 核对患者身份
    ├── 交代注意事项
    └── 发药签字
    │
    ↓
消耗登记
    │
    ├── 专账登记
    ├── 填写消耗记录
    └── 库存核对
    │
    ↓
处方存档
```
